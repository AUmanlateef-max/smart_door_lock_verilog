# Smart Door Lock System (Verilog)

This project implements a secure digital door lock using a **Finite State Machine (FSM)**. It is designed to demonstrate digital logic design and state-based control systems.

## Features
* **FSM-based Control:** Managed via states (IDLE, KEY_ENTRY, UNLOCK, ALARM).
* **Password Validation:** Compares user input against a preset code.
* **Testbench Included:** Verification of logic through comprehensive simulation.

## Tools Used
* **HDL:** Verilog
* **Simulation:** EDA Playground / Vivado
